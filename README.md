## サービス概要

少し遠回りのルートを提案し
普段行かない場所に寄り道して
近くを楽しむ自動 bot サービスです

## メインのターゲットユーザー

20 代の男女、独身、ひとりぐらし
日々仕事や学校に追われているがたまに息抜きがしたい人

## ユーザーが抱える課題

仕事終わりや休日に息抜きしたいもののどうすればいいかわからない
ふと思い立って近くのどこかに出掛けてみたいが下調べが面倒でなかなか行動ができない

## 解決方法

LINEbot という形式で直感的に現在地から目的地までの５~10 番目に早いルートを割り出し、その途中駅や乗り換え駅のカフェやレストラン、観光地を表示することで寄り道が楽しめるようにする

## 実装予定の機能

- 路線検索機能
  - 駅すぱあと API を使い(予定)路線検索をして、そのなかの 6~10 番目程度の早さのルートをランダムで抽出し表示
- おすすめ検索
  - 路線検索で出てきた途中駅の駅名でいくつかの選択肢で検索(カフェ、レストラン、居酒屋、観光など)してその検索結果を表示

## なぜこのサービスを作りたいか

めんどくさがりやだけどどこかに行きたい、あまり時間がないけど息抜きしたいがいちいちインターネットで場所を検索することが億劫でありできるだけ効率的で簡単に寄り道できるサイトがあったらいいなと思ったから。
